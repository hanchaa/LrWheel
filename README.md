# LrWheel
라이트룸 슬라이더를 조절 휠 제작 프로젝트

ESP32 보드를 사용해 블루투스 입력으로 라이트룸 단축키 입력

-------------------------------------
Project that makes wheel to control lightroom slider

Using ESP32 board to input shortcut key by wireless
